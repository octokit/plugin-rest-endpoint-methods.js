# List team projects

Lists the organization projects for a team.

**Note:** You can also specify a team by `org_id` and `team_id` using the route `GET /organizations/{org_id}/team/{team_id}/projects`.

```js
octokit.rest.teams.listProjectsInOrg({
  org,
  team_slug,
});
```
