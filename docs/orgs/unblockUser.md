# Unblock a user from an organization

```js
octokit.rest.orgs.unblockUser({
  org,
  username,
});
```

      <th>name</th>
      <th>required</th>

    <tr><td>org</td><td>yes</td><td>

The organization name. The name is not case sensitive.

</td></tr>

The handle for the GitHub user account.

</td></tr>
