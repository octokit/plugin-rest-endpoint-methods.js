# Set primary email visibility for the authenticated user

Sets the visibility for your primary email addresses.

```js
octokit.rest.users.setPrimaryEmailVisibilityForAuthenticatedUser({
  visibility,
});
```

    <tr>
      <th>name</th>
      <th>required</th>

    </tr>
  </thead>

</td></tr>
