# Get emojis

Lists all the emojis available to use on GitHub.

```js
octokit.emojis.get();
```

## Parameters

This endpoint has no parameters

See also: [GitHub Developer Guide documentation](https://developer.github.com/v3/emojis/#get-emojis).
